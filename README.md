# DevelopersHub Flutter Internship - Week 1 Task
**Internship ID: DHC-5322**

This is my project for the first week of the mobile development internship. The goals for this week were setting up the software, making a simple layout, and learning how to move between different screens.

## What I Did This Week
* **Set Up Everything:** I installed Flutter and set up Android Studio to run the app on my phone.
* **Made a Login Screen:** Built a user interface with two text fields where users can type their email and password.
* **Added a Button and Links:** Put in a "Forgot Password?" button and a blue "Login" button at the bottom.
* **Screen Navigation:** Created a second screen (Home Screen) that opens automatically when you click the Login button.
* **Simple Validation:** Added checking logic to show an error message if the fields are left empty or if the email does manager to miss the "@" symbol.

## Layout Widgets Used
To build the look of this app, I used basic Flutter widgets:
* **Container:** To add padding and space around the edge of the screen.
* **Column:** To stack the email field, password field, and buttons from top to bottom.
* **Row:** To place the "Forgot Password?" link neatly to the right side.
* **SizedBox:** To create empty gaps between the text boxes so they don't touch.

## How to Open and Run It
1. Open **Android Studio**.
2. Open this project folder.
3. Connect your Android phone with a USB cable (or open an emulator).
4. Press the green **Run** button at the top to install it on your device.
